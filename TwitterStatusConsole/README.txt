Description

The Twitter Status Console allows a user to send and manipulate statuses on their timeline, provided that certain variables are met. These variables can be filled in through the details given to you after creating an applet in Twitter. For details on how to, please visit this site (https://apps.twitter.com/), hit the "Create New App" button (https://i.imgur.com/hhsY12s.png), fill out the form data (here is some test data (https://i.imgur.com/fILbMi4.png) if you prefer, just change the name) then switch to the "Keys and Access Tokens" tab.


Prerequisites:

Python 3.6
Git (Latest)
The Tweepy module (pip install git+git://github.com/tweepy/tweepy.git)
